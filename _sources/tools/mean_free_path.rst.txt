Mean free path calculator
=========================

The present tool evaluates the mean free path of one or more
species in a gas mixture from the VSS collision parameters of
each species, and reports the DSMC numerical bounds
:math:`\Delta x \lesssim \lambda/3` and
:math:`\Delta t \lesssim \tau_c/3` that the mesh and time-step
must respect (see :doc:`../theory/foundations`). The tool exposes
both the pure-species table (one line per species) and the
**pair table** (one line per active collision pair), so that the
cross parameters :math:`(d_{st}, \omega_{st})` that govern the
binary interactions can be edited independently —  because the
rigorous VSS mixture calculation treats the cross terms as
**distinct fitted parameters**, not as means of the pure-species
values.

A short derivation, the combining-rule convention used to
initialise the pair table, and a discussion of the cases in which
the combining rule is inadequate are given at the end of the
page. The calculator runs entirely in the browser: no input is
sent to any server.

     <details style="margin-top:1.2em;">
       <summary>Show the formula used</summary>
       <p>
         The mean free path of species <em>s</em> in a mixture is
         the reciprocal of the sum of its collision frequencies with
         every species <em>t</em> in the mixture:
       </p>
       <p class="mono">
         λ<sub>s</sub> = 1 / Σ<sub>t</sub> &nbsp; n<sub>t</sub> · π · d<sub>st</sub><sup>2</sup> · (T<sub>ref</sub> / T)<sup>ω<sub>st</sub> − 1/2</sup> · (1 + m<sub>s</sub>/m<sub>t</sub>)<sup>1/2</sup>
       </p>
       <p>
         The cross parameters <em>d<sub>st</sub></em> and
         <em>ω<sub>st</sub></em> are taken from the <strong>pair
         table</strong> (section 4). When the pair entry has not been
         overridden, they are initialised from the arithmetic-mean
         combining rule,
         <span class="mono">d<sub>st</sub> = (d<sub>s</sub>+d<sub>t</sub>)/2</span> and
         <span class="mono">ω<sub>st</sub> = (ω<sub>s</sub>+ω<sub>t</sub>)/2</span>,
         which is not rigorous —  the background section at the bottom
         of the page discusses when this approximation is adequate
         and when it is not. The reference temperature is
         <span class="mono">T<sub>ref</sub> = 273 K</span>, the
         per-species number density is
         <span class="mono">n<sub>t</sub> = x<sub>t</sub> · p / (k<sub>B</sub> T)</span>,
         and the mean thermal speed is
         <span class="mono">c̄<sub>s</sub> = √(8 k<sub>B</sub> T / π m<sub>s</sub>)</span>,
         which yields the mean collision time
         <span class="mono">τ<sub>c,s</sub> = λ<sub>s</sub> / c̄<sub>s</sub></span>.
       </p>
       <p>
         Note the asymmetry of the pair factor
         <span class="mono">(1 + m<sub>s</sub>/m<sub>t</sub>)<sup>1/2</sup></span>:
         <em>s</em> and <em>t</em> are <strong>not</strong>
         interchangeable in this formula. The self-contribution
         (<em>t = s</em>) recovers the factor √2 correctly, since
         <span class="mono">(1 + m<sub>s</sub>/m<sub>s</sub>)<sup>1/2</sup> = √2</span>.
       </p>
     </details>

       'Air (30 km, 227 K, 12 hPa)':             { T: 226.51, p: 1171.9,  mix: { N2: 0.78, O2: 0.21, Ar: 0.01 } },
       'Air (80 km, 199 K, 1 Pa)':               { T: 198.6,  p: 1.052,   mix: { N2: 0.78, O2: 0.21, Ar: 0.01 } },
       'Pure N2 at 300 K, 1 atm':                { T: 300,    p: 101325,  mix: { N2: 1.0 } },

       'Pure CO2 at 300 K, 1 atm (Mars atm.)':   { T: 210,    p: 700,     mix: { CO2: 1.0 } },
       'Re-entry shock layer (10 kK, 100 Pa)':   { T: 10000,  p: 100,     mix: { N2: 0.55, N: 0.15, O2: 0.05, O: 0.20, NO: 0.05 } },
       'Heat-bath 0-D (20 kK, 1 atm, O2)':       { T: 20000,  p: 101325,  mix: { O2: 1.0 } }
     };

     // Pair override map. Key is canonical "A|B" (sorted alphabetically).
     // Value is { d: metres, omega: number } when the pair has been
     // manually overridden; absent otherwise.
     const pairOverride = {};

     function pairKey(s, t) {
       return [s, t].sort().join('|');
     }

     function fmtSci(x, digits) {
       if (!isFinite(x)) return '—';
       return x.toExponential(digits || 3);
     }

     function renderPresets() {
       const el = document.getElementById('presets');
       let html = '';
       for (const name in PRESETS) {
         html += `<span class="preset" data-name="${name}">${name}</span>`;
       }
       el.innerHTML = html;
       el.querySelectorAll('.preset').forEach(btn => {
         btn.addEventListener('click', () => loadPreset(btn.dataset.name));
       });
     }

     function renderSpeciesList() {
       let html = '';
       for (const s in SPECIES_DB) {
         const sp = SPECIES_DB[s];
         html += `
           <tr data-species="${s}">
             <td><strong>${s}</strong></td>
             <td>${sp.type}</td>
             <td><input class="param mass"  type="number" step="any" value="${sp.m.toExponential(4)}"></td>
             <td><input class="param diam"  type="number" step="any" value="${(sp.d*1e10).toFixed(3)}"></td>
             <td><input class="param omega" type="number" step="0.01" value="${sp.omega.toFixed(2)}"></td>
             <td><input class="mole-frac"   type="number" step="any" min="0" value="0"></td>
           </tr>`;
       }
       document.getElementById('species-list').innerHTML = html;
     }

     function currentSpeciesParams() {
       // Returns { s: { m, d, omega, x } } — all species in SPECIES_DB order,
       // with the current table values, regardless of x.
       const out = {};
       document.querySelectorAll('#species-list tr').forEach(row => {
         const s = row.dataset.species;
         out[s] = {
           m:     parseFloat(row.querySelector('input.mass').value),
           d:     parseFloat(row.querySelector('input.diam').value) * 1e-10,
           omega: parseFloat(row.querySelector('input.omega').value),
           x:     parseFloat(row.querySelector('input.mole-frac').value) || 0
         };
       });
       return out;
     }

     function renderPairList() {
       const params = currentSpeciesParams();
       const keys = Object.keys(params);
       let html = '';
       for (let i = 0; i < keys.length; i++) {
         for (let j = i; j < keys.length; j++) {
           const s = keys[i], t = keys[j];
           const ps = params[s], pt = params[t];
           const active = ps.x > 0 && pt.x > 0;
           const k = pairKey(s, t);
           const ovr = pairOverride[k];
           const d_rule  = 0.5 * (ps.d + pt.d);
           const w_rule  = 0.5 * (ps.omega + pt.omega);
           const d_used  = ovr ? ovr.d     : d_rule;
           const w_used  = ovr ? ovr.omega : w_rule;
           const massRatio = ps.m / pt.m;
           html += `
             <tr class="pair-row${active ? '' : ' inactive'}${ovr ? ' overridden' : ''}"
                 data-s="${s}" data-t="${t}">
               <td><strong>${s}</strong> / <strong>${t}</strong></td>
               <td><input class="param pair-d"     type="number" step="any" value="${(d_used*1e10).toFixed(4)}"></td>
               <td><input class="param pair-omega" type="number" step="0.01" value="${w_used.toFixed(3)}"></td>
               <td>${massRatio.toExponential(3)}</td>
               <td>${ovr ? 'overridden' : 'from combining rule'}</td>
             </tr>`;
         }
       }
       document.getElementById('pair-list').innerHTML = html;

       // Attach listeners to pair inputs: on change, mark override
       document.querySelectorAll('#pair-list tr').forEach(row => {
         const s = row.dataset.s, t = row.dataset.t;
         row.querySelectorAll('input.pair-d, input.pair-omega').forEach(inp => {
           inp.addEventListener('change', () => {
             const d = parseFloat(row.querySelector('input.pair-d').value) * 1e-10;
             const w = parseFloat(row.querySelector('input.pair-omega').value);
             pairOverride[pairKey(s, t)] = { d, omega: w };
             renderPairList();   // rerender to show "overridden" status
             computeMFP();
           });
         });
       });
     }

     function resetPairTableToRule() {
       for (const k in pairOverride) delete pairOverride[k];
       renderPairList();
       computeMFP();
     }

     function loadPreset(name) {
       const p = PRESETS[name];
       if (!p) return;
       document.getElementById('T').value = p.T;
       document.getElementById('p').value = p.p;
       document.querySelectorAll('#species-list tr').forEach(row => {
         row.querySelector('input.mole-frac').value = '0';
       });
       for (const s in p.mix) {
         const row = document.querySelector(`#species-list tr[data-species="${s}"]`);
         if (row) row.querySelector('input.mole-frac').value = p.mix[s];
       }
       renderPairList();
       computeMFP();
     }

     function normaliseMoleFractions() {
       const params = currentSpeciesParams();
       let total = 0;
       for (const s in params) total += params[s].x;
       if (total <= 0) {
         document.getElementById('warn').textContent = 'Cannot normalise: all mole fractions are zero.';
         return;
       }
       document.querySelectorAll('#species-list tr').forEach(row => {
         const input = row.querySelector('input.mole-frac');
         const x = parseFloat(input.value) || 0;
         if (x > 0) input.value = (x / total).toFixed(6);
       });
       document.getElementById('warn').textContent = '';
       renderPairList();
       computeMFP();
     }

     function activeSpeciesParams() {
       // Returns only those species with x > 0.
       const all = currentSpeciesParams();
       const out = {};
       for (const s in all) if (all[s].x > 0) out[s] = all[s];
       return out;
     }

     function pairUsed(s, t, all) {
       const ovr = pairOverride[pairKey(s, t)];
       if (ovr) return { d: ovr.d, omega: ovr.omega, overridden: true };
       return {
         d:     0.5 * (all[s].d + all[t].d),
         omega: 0.5 * (all[s].omega + all[t].omega),
         overridden: false
       };
     }

     function computeMFP() {
       const warn = document.getElementById('warn');
       warn.textContent = '';

       const T = parseFloat(document.getElementById('T').value);
       const p = parseFloat(document.getElementById('p').value);
       const L = parseFloat(document.getElementById('L').value);

       if (!(T > 0) || !(p > 0)) {
         warn.textContent = 'Temperature and pressure must both be strictly positive.';
         document.getElementById('results').innerHTML = '';
         return;
       }

       const n_tot = p / (K_B * T);

       const mix = activeSpeciesParams();
       const species = Object.keys(mix);
       if (species.length === 0) {
         warn.textContent = 'Please set a non-zero mole fraction for at least one species.';
         document.getElementById('results').innerHTML = '';
         return;
       }

       let total = 0;
       species.forEach(s => total += mix[s].x);
       if (Math.abs(total - 1.0) > 1e-3) {
         warn.textContent = `Mole fractions sum to ${total.toFixed(4)}, not 1. Click "Normalise" to rescale.`;
       }

       const perSpecies = {};
       for (const s of species) {
         const ps = mix[s];
         let invLambda = 0;
         for (const t of species) {
           const pt = mix[t];
           const n_t = pt.x * n_tot;
           const pair = pairUsed(s, t, mix);
           const massFactor = Math.sqrt(1 + ps.m / pt.m);
           const tempFactor = Math.pow(T_REF / T, pair.omega - 0.5);
           invLambda += n_t * Math.PI * pair.d * pair.d * tempFactor * massFactor;
         }
         const lambda_s = 1.0 / invLambda;
         const c_bar_s  = Math.sqrt(8 * K_B * T / (Math.PI * ps.m));
         const tau_c_s  = lambda_s / c_bar_s;
         perSpecies[s] = { lambda: lambda_s, c_bar: c_bar_s, tau_c: tau_c_s };
       }

       let minLambda = Infinity, minLambdaSp = '';
       let minTau    = Infinity, minTauSp    = '';
       for (const s in perSpecies) {
         if (perSpecies[s].lambda < minLambda) { minLambda = perSpecies[s].lambda; minLambdaSp = s; }
         if (perSpecies[s].tau_c  < minTau   ) { minTau    = perSpecies[s].tau_c;  minTauSp    = s; }
       }

       let html = '<h4>Per-species results</h4><table><thead><tr>';
       html += '<th>Species</th><th>n<sub>s</sub> [m⁻³]</th><th>λ<sub>s</sub> [m]</th><th>c̄<sub>s</sub> [m/s]</th><th>τ<sub>c,s</sub> [s]</th>';
       html += '</tr></thead><tbody>';
       for (const s in perSpecies) {
         const r = perSpecies[s];
         const ns = mix[s].x * n_tot;
         html += `<tr><td><strong>${s}</strong></td><td>${fmtSci(ns)}</td><td>${fmtSci(r.lambda)}</td><td>${r.c_bar.toFixed(1)}</td><td>${fmtSci(r.tau_c)}</td></tr>`;
       }
       html += '</tbody></table>';

       html += '<h4>DSMC numerical constraints</h4><table>';
       html += `<tr><td>Total number density n</td><td>${fmtSci(n_tot)} m⁻³</td></tr>`;
       html += `<tr><td>Minimum mean free path λ<sub>min</sub></td><td>${fmtSci(minLambda)} m (${minLambdaSp})</td></tr>`;
       html += `<tr class="constraint"><td><strong>Recommended Δx ≤ λ<sub>min</sub> / 3</strong></td><td><strong>${fmtSci(minLambda/3)} m</strong></td></tr>`;
       html += `<tr><td>Minimum mean collision time τ<sub>c,min</sub></td><td>${fmtSci(minTau)} s (${minTauSp})</td></tr>`;
       html += `<tr class="constraint"><td><strong>Recommended Δt ≤ τ<sub>c,min</sub> / 3</strong></td><td><strong>${fmtSci(minTau/3)} s</strong></td></tr>`;

       if (isFinite(L) && L > 0) {
         const Kn = minLambda / L;
         let regime = '';
         if      (Kn < 0.01) regime = 'Continuum — Navier–Stokes adequate';
         else if (Kn < 0.1)  regime = 'Slip — NS with slip conditions';
         else if (Kn < 10)   regime = 'Transitional — DSMC is the method of choice';
         else                regime = 'Free molecular';
         const nCells1D = Math.ceil(L / (minLambda / 3));
         html += `<tr><td>Knudsen number based on L = ${L} m</td><td><strong>Kn = ${fmtSci(Kn)}</strong> — ${regime}</td></tr>`;
         html += `<tr><td>Cells along L at Δx = λ<sub>min</sub>/3</td><td>≈ ${nCells1D.toLocaleString()} (1-D); scale by L² or L³ for 2-D / 3-D</td></tr>`;
       }

       html += '</table>';

       // Count overridden pair entries used in this computation
       let nOverrides = 0;
       for (const s of species) for (const t of species) {
         if (s <= t && pairOverride[pairKey(s, t)]) nOverrides++;
       }
       if (nOverrides > 0) {
         html += `<p class="notice"><em>${nOverrides} pair entr${nOverrides===1?'y':'ies'} overridden in the pair table above.</em></p>`;
       }

       document.getElementById('results').innerHTML = html;
     }

     function onSpeciesEdited() {
       // A species-level edit: rerender the pair table (preserving overrides)
       // and recompute.
       renderPairList();
       computeMFP();
     }

       renderPresets();
       renderSpeciesList();
       ['T','p','L'].forEach(id => {
         document.getElementById(id).addEventListener('input', computeMFP);
       });
       document.getElementById('species-list').addEventListener('input', onSpeciesEdited);
       loadPreset('Air (surface, 288 K, 1 atm)');

     function speciesAt(alt, p_Pa, T_K, mixFrac) {
       // Build a { s: { m, d, omega, x } } map for this altitude,
       // reading d/omega/m from the current table rows (so user edits
       // to pure-species parameters propagate into the profile). Pair
       // parameters use the current pair-table values too.
       const all = currentSpeciesParams();
       const out = {};
       for (const s in mixFrac) {
         if (!(s in all)) continue;
         out[s] = { m: all[s].m, d: all[s].d, omega: all[s].omega, x: mixFrac[s] };
       }
       return out;
     }

     function lambdaForMix(mix, n_tot, T_K) {
       const species = Object.keys(mix).filter(s => mix[s].x > 0);
       if (species.length === 0) return { lambda: NaN, tau: NaN };
       let minLambda = Infinity, minTau = Infinity;
       for (const s of species) {
         const ps = mix[s];
         let invLambda = 0;
         for (const t of species) {
           const pt = mix[t];
           const n_t = pt.x * n_tot;
           const pair = pairUsed(s, t, mix);
           const massFactor = Math.sqrt(1 + ps.m / pt.m);
           const tempFactor = Math.pow(T_REF / T_K, pair.omega - 0.5);
           invLambda += n_t * Math.PI * pair.d * pair.d * tempFactor * massFactor;
         }
         const lam = 1.0 / invLambda;
         const cbar = Math.sqrt(8 * K_B * T_K / (Math.PI * ps.m));
         const tau = lam / cbar;
         if (lam < minLambda) minLambda = lam;
         if (tau < minTau) minTau = tau;
       }
       return { lambda: minLambda, tau: minTau };
     }

     function regimeLabel(Kn) {
       if (!isFinite(Kn)) return '—';
       if (Kn < 0.01) return 'Continuum';
       if (Kn < 0.1)  return 'Slip';
       if (Kn < 10)   return 'Transitional';
       return 'Free molecular';
     }

Background
----------

Derivation
``````````

Consider a single molecule of species :math:`s` moving through a
gas mixture. Its collision frequency with species :math:`t` is
the product of the target number density :math:`n_t`, the binary
collisional cross section :math:`\sigma_{st}(g)`, and the mean
relative speed :math:`\bar{g}_{st}`:

.. math::

   \nu_{st}
     = n_t\,\sigma_{st}(g)\,\bar{g}_{st},
   \qquad
   \bar{g}_{st}
     = \sqrt{\frac{8\,k_B T}{\pi\,\mu_{st}}},

with :math:`\mu_{st} = m_s m_t / (m_s + m_t)` the reduced mass.
Summing over target species and dividing by the thermal speed of
species :math:`s` yields

.. math::

   \lambda_s
     = \frac{\bar{c}_s}{\displaystyle \sum_t n_t\,\sigma_{st}\,\bar{g}_{st}},
   \qquad
   \bar{c}_s = \sqrt{\frac{8\,k_B T}{\pi\,m_s}}.

For a VSS cross section
:math:`\sigma_{st}(g) = \pi\,d_{st}^{2}\,(T_{\mathrm{ref}}/T)^{\omega_{st} - 1/2}
/\Gamma(5/2 - \omega_{st})`,
the :math:`\Gamma` factor and the thermal-speed ratio combine to
give

.. math::

   \lambda_s
     = \frac{1}
            {\displaystyle\sum_{t}
               n_t\,\pi\,d_{st}^{2}\,
               \left(\tfrac{T_{\mathrm{ref}}}{T}\right)^{\omega_{st} - 1/2}\,
               \left(1 + \tfrac{m_s}{m_t}\right)^{1/2}},

which is the expression evaluated by the calculator. Note in
particular:

* The inner sum is a **sum over the target species** :math:`t`,
  not a weighted average. Each term contributes additively
  to the collision frequency.
* The mass factor :math:`(1 + m_s/m_t)^{1/2}` is
  **asymmetric** in :math:`(s, t)`: it is not the same as
  :math:`(1 + m_t/m_s)^{1/2}`. The self-contribution
  :math:`t = s` gives :math:`\sqrt{2}`, recovering the classic
  single-species result.
* The pair parameters :math:`d_{st}` and :math:`\omega_{st}`
  are in general **fitted independently** for each binary pair,
  from viscosity, diffusion, and thermal-conductivity data.
  They are not derived from the pure-species values through any
  mean.

Combining rules — when they apply, when they do not
```````````````````````````````````````````````````

When independent pair fits are not available, a **combining
rule** is used to generate :math:`d_{st}` and :math:`\omega_{st}`
from the pure-species values :math:`d_s, d_t, \omega_s,
\omega_t`. The calculator initialises the pair table through
Bird's arithmetic-mean rule [Bird1994m]_,

.. math::

   d_{st} = \tfrac{1}{2}\,(d_s + d_t),
   \qquad
   \omega_{st} = \tfrac{1}{2}\,(\omega_s + \omega_t),

which is analogous to the Lorentz-Berthelot rule used for
Lennard-Jones mixtures and is also the default convention in
most DSMC solvers. The arithmetic-mean rule is adequate when:

* the species involved are chemically similar (same family,
  similar polarisability);
* the pure-species :math:`\omega_s` are close to each other
  (within :math:`\pm 0.1`);
* quantitative accuracy at the :math:`\sim\!10\%` level on
  transport coefficients is sufficient.

It becomes unreliable when:

* the colliding species differ greatly in mass or size
  (e.g. H\ :sub:`2` – CO\ :sub:`2`);
* one of the species is polar and the other is not, so that a
  weakly attractive interaction becomes dominant;
* a chemically reactive pair (e.g. O + NO) is involved, whose
  cross sections are better fitted from QCT calculations than
  from transport data.

For production DSMC work, the pair parameters should therefore
be obtained from:

* experimental binary diffusion or binary viscosity data (fit
  :math:`d_{st}, \omega_{st}` at the pair level);
* classical-trajectory calculations on the relevant
  potential-energy surface;
* the pair tables compiled in Koura and Matsumoto
  [Koura1991m]_ or Hirschfelder et al. [HCB1954m]_.

Any of these may be entered directly in the **pair table**
(section 4 of the calculator). Overridden entries are
highlighted and take precedence over the combining rule; they
persist when the pure-species values are edited, and are cleared
only by the *Reset pair table to combining rule* button.

DSMC numerical constraints
``````````````````````````

The DSMC convergence conditions of :doc:`../theory/foundations`
require that the spatial and temporal resolution respect the
local kinetic scales:

.. math::

   \Delta x \lesssim \frac{\lambda}{3},
   \qquad
   \Delta t \lesssim \frac{\tau_c}{3},

with :math:`\tau_c = \lambda / \bar{c}` the mean collision time.
Violating either bound introduces a bias in the collision
statistics that is difficult to diagnose *a posteriori*. In a
multi-species mixture, the binding constraint is the **smallest**
:math:`\lambda_s` across all species present —  typically the
heaviest, slowest species at the highest local pressure. The
calculator identifies this binding species and reports the
corresponding :math:`\Delta x` and :math:`\Delta t` bounds.

On the sub-cell machinery
`````````````````````````

When the flow spans several decades in pressure or temperature,
a fixed mesh cannot satisfy :math:`\Delta x < \lambda/3`
everywhere simultaneously. The solver's sub-cell adaptation
layer (:doc:`../classes/adaptation`) is the operational fix:
each cell is logically subdivided, on the fly, into a number of
sub-cells per direction that keeps the sub-cell size below the
local mean free path. The calculator's :math:`\Delta x`
recommendation therefore applies either to the mesh cell size
(in non-adaptive runs) or to the effective sub-cell size (in
adaptive runs).

Caveats
```````

* The preloaded VSS pure-species parameters are Bird's reference
  values at :math:`T_{\mathrm{ref}} = 273~\mathrm{K}`. More
  recent fits exist in the literature for individual species and
  species pairs; users are encouraged to override the
  pure-species entries and the pair entries directly in the
  tables if more appropriate values are available.
* The calculator assumes a **single-temperature** gas. Under
  thermal non-equilibrium (:math:`T_t \ne T_\mathrm{int}`) the
  mean free path depends only on the translational temperature
  and on the ground-state populations through :math:`n_s`; the
  numbers above therefore remain the right first-order estimate
  provided :math:`T` is read as the translational temperature.
* The Knudsen classification is the standard textbook one
  (continuum :math:`< 0.01`, slip :math:`< 0.1`, transitional
  :math:`< 10`, free molecular otherwise); it should be treated
  as indicative rather than sharp.

   potential," *Physics of Fluids A*, **3** (10), 2459–2465
   (1991).
